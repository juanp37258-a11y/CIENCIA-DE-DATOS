## 📚 Contenido del Curso

Este repositorio contiene clases prácticas sobre las herramientas fundamentales de la ciencia de datos:

### 📘 Clase 1 - Fundamentos
- **NumPy** — arrays, matrices, operaciones matemáticas
- **Juegos interactivos** — dados y cartas
- **Matplotlib** — gráficas básicas
- **Pandas** — Series y DataFrames

### 📗 Clase 2 - Manipulación de Datos
- **Series de Pandas** — índices personalizados
- **DataFrames** — filtros, selección con `loc` e `iloc`
- **Exportación de datos** — CSV, HTML, JSON

### 📊 Clase 3 - Visualización con Matplotlib
- **Gráficos de línea** — configuración y personalización
- **Carga de datos reales** — lectura de CSV
- **Exploración explorativa** — head(), describe(), info()
- **Histogramas** — distribución con línea de media
- **Boxplots** — análisis con Seaborn

### 📙 Clase 4 - Análisis Exploratorio Avanzado
- **Dataset de crédito bancario** — exploración completa
- **Análisis de variables categóricas** — género, contrato, ocupación, estado civil
- **Visualizaciones avanzadas** — histogramas y boxplots con Seaborn
- **Paletas de color** — uso de viridis y otras paletas

## 🚀 Cómo Usar Este Repositorio

1. **Clonar el repositorio:**
```bash
git clone https://github.com/juanp37258-a11y/CIENCIA-DE-DATOS.git
cd CIENCIA-DE-DATOS
```

2. **Instalar dependencias:**
```bash
pip install numpy pandas matplotlib seaborn
```

3. **Ejecutar una clase:**
```bash
python3 clase_1.py
```
